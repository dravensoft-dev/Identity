# Self-hosted Resource Policy Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bundle Arena's text fonts as git-tracked, self-hosted `.woff2` served from a single origin (`assets/fonts/` + `@font-face` in `tokens/fonts.css`), remove every font CDN request, delete the redundant Angular `fonts/` layer, and document icons as install-the-official-package-by-default.

**Architecture:** `assets/fonts/` becomes the single source of truth for the 14 `.woff2` binaries. `tokens/fonts.css` declares them once with `@font-face` using `url('../assets/fonts/…')`, which the browser resolves relative to `tokens/fonts.css` regardless of the importing file — so React (via `styles.css`), Angular (via `arena-tailwind.css` → `styles.css`), Tailwind, the guideline pages, and the `.dc.html` brand pages all inherit fonts from that one place. Icons stay unbundled; only the docs change to state the default-install policy.

**Tech Stack:** Plain CSS `@font-face`, static HTML, a Node ESM download script (`scripts/fetch-fonts.mjs`). No build, no test runner, no package.json — this repo is served statically over `python3 -m http.server`.

## Global Constraints

- **English only** — all code, comments, docs, UI copy.
- **Font families and weights are frozen** — Archivo (400,500,600,700,800,900), Familjen Grotesk (400,500,600,700), Spline Sans Mono (400,500,600,700). Do not add, drop, or re-subset. They must stay identical to `tokens/typography.css`.
- **`url()` paths are relative to `tokens/fonts.css`** — always `../assets/fonts/<file>.woff2`.
- **The `.woff2` binaries are git-tracked** — Arena ships them; they are no longer git-ignored.
- **Do not edit the frozen `## [3.1.0]` CHANGELOG entry** — new work lands under `## [Unreleased]` on top.
- **Do not edit `support.js`** — it is a generated bundle.
- **No version bump / tag in this plan** — cutting the release (plugin.json, marketplace.json, README header, CHANGELOG rename, `source.ref`, tag, `check-release.mjs`) is a separate, user-triggered step.
- **Verification is grep + HTTP inspection**, not a test runner. Serve with `python3 -m http.server 8000` from the repo root.

- [ ] **Step 1: Create `scripts/fetch-fonts.mjs`**

```javascript
/* scripts/fetch-fonts.mjs
   Downloads the latin-subset woff2 binaries for Arena's three families into
   assets/fonts/ and (re)generates tokens/fonts.css with local url()s only —
   no CDN request, CSP-clean. assets/fonts/ is the single source of truth,
   shared by every framework layer. Run: node scripts/fetch-fonts.mjs */
import { writeFileSync, mkdirSync } from 'node:fs';
import { dirname, join } from 'node:path';
import { fileURLToPath } from 'node:url';

const here = dirname(fileURLToPath(import.meta.url));
const fontsDir = join(here, '..', 'assets', 'fonts');
const cssPath = join(here, '..', 'tokens', 'fonts.css');
mkdirSync(fontsDir, { recursive: true });

/* A modern-browser UA makes Google Fonts serve woff2 (not ttf). */
const UA =
  'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 ' +
  '(KHTML, like Gecko) Chrome/120.0 Safari/537.36';

const FAMILIES = [
  { css: 'Archivo',           slug: 'archivo',           weights: [400, 500, 600, 700, 800, 900] },
  { css: 'Familjen Grotesk',  slug: 'familjen-grotesk',  weights: [400, 500, 600, 700] },
  { css: 'Spline Sans Mono',  slug: 'spline-sans-mono',  weights: [400, 500, 600, 700] },
];

async function google(css, weights) {
  const family = `${css.replace(/ /g, '+')}:wght@${weights.join(';')}`;
  const url = `https://fonts.googleapis.com/css2?family=${family}&display=swap`;
  const res = await fetch(url, { headers: { 'User-Agent': UA } });
  if (!res.ok) throw new Error(`CSS fetch failed for ${css}: ${res.status}`);
  return res.text();
}

/* Pull the latin-subset @font-face blocks: each carries a font-weight and a
   woff2 src url and is tagged by Google Fonts with a "latin" subset comment. */
function latinFaces(cssText) {
  const faces = [];
  const re = /\/\*\s*latin\s*\*\/\s*@font-face\s*{([^}]*)}/g;
  let m;
  while ((m = re.exec(cssText)) !== null) {
    const block = m[1];
    const weight = /font-weight:\s*(\d+)/.exec(block)?.[1];
    const src = /src:\s*url\(([^)]+\.woff2)\)/.exec(block)?.[1];
    if (weight && src) faces.push({ weight: Number(weight), src });
  }
  return faces;
}

async function download(url) {
  const res = await fetch(url, { headers: { 'User-Agent': UA } });
  if (!res.ok) throw new Error(`binary fetch failed: ${url} ${res.status}`);
  return Buffer.from(await res.arrayBuffer());
}

const rules = [];
for (const fam of FAMILIES) {
  const cssText = await google(fam.css, fam.weights);
  const faces = latinFaces(cssText);
  for (const face of faces) {
    const file = `${fam.slug}-${face.weight}.woff2`;
    writeFileSync(join(fontsDir, file), await download(face.src));
    rules.push(
      `@font-face {\n` +
        `  font-family: '${fam.css}';\n` +
        `  font-style: normal;\n` +
        `  font-weight: ${face.weight};\n` +
        `  font-display: swap;\n` +
        `  src: url('../assets/fonts/${file}') format('woff2');\n` +
        `}`
    );
  }
  console.log(`${fam.css}: ${faces.length} weights`);
}

const header =
  `/* tokens/fonts.css — self-hosted, CSP-clean webfonts.\n` +
  `   GENERATED by scripts/fetch-fonts.mjs; edit that, not this. Binaries live in\n` +
  `   assets/fonts/. Families match tokens/typography.css (Archivo / Familjen\n` +
  `   Grotesk / Spline Sans Mono). */\n`;
writeFileSync(cssPath, `${header}\n${rules.join('\n\n')}\n`);
console.log('wrote tokens/fonts.css');
```
